# groovy-web-maven-spring-thymeleaf-hello

## Description
A POC for springboot web app with thymeleaf support.

## Tech stack
- scala
- sbt
- scala
- sbt
- groovy
- maven
  - springboot
  - thymeleaf
  - bootstrap

## Docker stack
- hseeberger/scala-sbt:11.0.2-oraclelinux7_1.3.5_2.12.10
- hseeberger/scala-sbt:11.0.2-oraclelinux7_1.3.5_2.12.10
- openjdk:11-jdk

## To run
`sudo ./install.sh -u`
Available at http://localhost

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
